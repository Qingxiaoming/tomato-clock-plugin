# 番茄钟 (Tomato Clock)

[English](README.md)

一个功能完整的 Obsidian 计时插件，支持番茄钟、正计时、倒计时三种模式，提供类 Toggl Track 的时间线视图与项目统计。所有数据保存在本地 vault 中。

## 功能概览

### 三种计时模式

- **番茄钟 (Pomodoro)**：标准番茄工作法循环。可配置工作时长、短休息、长休息与每组循环数。自动切换工作/休息阶段。
- **正计时 (Stopwatch)**：从 0 开始向上累计，适合无明确时间限制的任务。
- **倒计时 (Countdown)**：从设定分钟数向下倒计时，适合限时任务。

### 双面板系统

**小面板（Compact）** —— 默认侧边栏视图，极简紧凑：
- 顶部：**项目选择器** + **任务输入框**
- **当前时间行**：大字号当前时间 + 年/月/日/周 **周期性笔记指示点** + **模式切换按钮**
- **今日时间线**：灰色轨道 + 彩色色块（各时段记录）+ 紫色**当前时刻竖线**
- **计时器行**：**阶段圆点列**（竖排，表示第几个番茄钟）| **时钟**（大字体，双击打开大面板，右键呼出模式操作菜单）| **操作按钮**（播放/跳过/重置/停止）
- **信息行**：当前状态（左）+ **今日总时长**（右）
- **日历区**：右侧导航（年月标签 + 上下翻页箭头）+ 左侧 **calendar-extended 嵌入式月历**（带日记标记）

**大面板（Full）** —— 双击小面板时钟打开，信息全面：
- 头部：标题 + 今日已完成番茄数
- 模式切换按钮（横排文字按钮）
- 项目选择器 + 任务输入框 + 倒计时分钟输入
- 番茄进度点（4 个一组）
- 大控制按钮（开始/暂停/继续/停止/跳过/重置）
- 四个标签页：**日历** / **列表** / **时间表** / **统计**

### 日历标签页

- **日/周/月** 三种视图模式，通过导航行切换
- 横向 00:00–24:00 时间轴，彩色条表示各时段记录
- 左侧时间标尺（00:00 ~ 24:00）
- 顶部**项目条**（Project Bar）显示当前视图的项目分布
- 右键点击彩色条可**编辑该条目的项目、任务、时长**
- 日期导航：前一天 / 后一天 / 今天

### 列表标签页

- 按日期分组的记录列表
- 显示每条记录的开始时间、结束时间、时长、模式、项目、任务名

### 时间表标签页

- 周视图横向时间轨道，类 Toggl Track 风格
- 日期导航 + 图例 + 总时长汇总

### 统计标签页

- 周期切换：日 / 周 / 月 / 年
- 总时长 + 番茄钟数量
- 项目分布（带颜色圆点）
- 柱状图趋势（周视图显示每日、月视图显示每周、年视图显示每月）
- **生成报告**：点击按钮可在日志文件夹下的 `Reports/` 子文件夹中生成对应的 Markdown 报告（周报/月报/年报）

### 嵌入式日历（小面板）

- 基于 calendar-extended 的月历视图
- 带日记标记、周期性笔记标记
- 点击年月标签可跳转回当前月
- 上下箭头翻页

### 日志系统

- **按天文件**：在用户指定的日志文件夹中，每天生成一个 `YYYY-MM-DD.md` 文件
- **格式**：`- 14:30 ~ 14:55 (25m) [pomodoro] 任务名称`
- **项目解析**：任务文本中若包含 `tomato_project: 项目名` 或 `tomato_project：项目名`，统计时会自动归类到该项目
- **日记链接**：可选在日志文件顶部自动插入当日日记的 `[[wikilink]]`（通过 Obsidian 核心「日记」插件获取路径）
- **跨天处理**：跨天的记录写入开始日期当天的文件，统计时通过代码逻辑补全
- **自动建文件夹**：若日志文件夹不存在，首次写入时自动创建
- **完成时自动打开**：可选在计时完成后自动在相邻窗格打开当日日志文件

### 项目管理

- 在设置面板中添加、编辑、删除项目
- 每个项目可配置专属颜色
- 大小面板均可选择当前项目，选择后计入日志与统计
- 时间线、日历、统计视图按项目颜色分段/分组显示

### 多语言

- 支持 **中文 / 英文** 切换
- 所有面板文本、设置项、通知内容均已国际化

### 状态栏

- 在 Obsidian 底部状态栏显示剩余/已用时间
- 三种显示模式：**完整模式**（显示时间和状态）、**简洁模式**（只显示是否计时）、**隐藏**
- 点击状态栏可直接打开小面板

### 通知与音效

- 阶段结束时触发系统通知（即使 Obsidian 在后台也能收到）
- 提示音通过 Web Audio API 在浏览器内合成，无需外部音频文件
- 首次加载时自动请求浏览器通知权限

### 会话恢复

- 计时器运行期间，每 10 秒自动保存恢复数据到 `recovery.json`
- 退出 Obsidian 前若计时器正在运行，会触发保存并提示防止意外关闭
- 重新打开 Obsidian 后自动读取 `recovery.json`，恢复之前的计时状态（模式、阶段、已用时间、项目、任务等）

### 命令面板

提供以下命令，可通过 Ctrl/Cmd+P 调用：
- 番茄钟：开始 / 暂停
- 番茄钟：重置
- 番茄钟：打开面板
- 番茄钟：切换到番茄钟
- 番茄钟：切换到正计时
- 番茄钟：切换到倒计时

## 设置项

| 选项 | 默认值 | 说明 |
|---|---|---|
| 语言 | 中文 | 界面语言（中文 / English） |
| 工作时长 | 25 分钟 | 每个番茄的工作时间 |
| 短休息 | 5 分钟 | 每个番茄后的短暂休息 |
| 长休息 | 15 分钟 | 完成一组循环后的长休息 |
| 每组循环数 | 4 | 触发长休息前的工作次数 |
| 倒计时时长 | 25 分钟 | 倒计时模式的默认时长 |
| 自动开始下一阶段 | 开 | 自动切换工作与休息 |
| 声音提醒 | 开 | 阶段结束时播放提示音 |
| 系统通知 | 开 | 后台使用时弹出系统通知 |
| 状态栏模式 | 完整模式 | 完整 / 简洁 / 隐藏 |
| 日志文件夹 | Tomato Logs | 每日日志文件的存放文件夹 |
| 链接当日日记 | 开 | 在日志文件顶部插入日记链接 |
| 完成时打开日志 | 开 | 计时完成后自动打开当日日志 |
| 日历吸附分钟数 | 5 分钟 | 日历时隙对齐粒度（1/5/10/15/30） |
| 小面板当前时间字号 | 1.7 rem | 当前时间行的字体大小 |
| 小面板计时器字号 | 1.8 rem | 时钟区域的字体大小 |
| 小面板当前时间字体 | Courier New | 当前时间行字体 |
| 小面板计时器字体 | Courier New | 时钟区域字体 |
| 项目 | — | 添加/编辑/删除项目及颜色 |
| 日历扩展设置 | — | calendar-extended 相关设置（周起始日、日记格式等） |

## 使用方法

1. 点击左侧工具栏的 ⏱ 图标（或执行「番茄钟：打开面板」命令）打开小面板。
2. 在小面板顶部的下拉框选择项目，输入本次任务名称。
3. 点击「开始」按钮开始计时。番茄钟模式下会自动循环工作/休息阶段。
4. 双击小面板的时钟区域可打开大面板，查看日历、列表、时间表与统计。
5. 计时完成后，日志自动追加到当天的日志文件中。若开启了「链接当日日记」，可通过反向链接在日记中找到对应日志。

## 安装

将仓库中的 `main.js`、`manifest.json` 和 `styles.css` 复制到你的 vault 目录 `.obsidian/plugins/tomato-clock/` 下，然后在 Obsidian 设置 → 社区插件中启用「番茄钟」。

## 隐私与数据

所有数据均保存在本地：
- 计时记录写入 vault 内的 Markdown 日志文件
- 恢复数据保存在插件目录下的 `recovery.json`
- 不连接任何外部服务器，无遥测或分析

## 技术栈

- TypeScript
- Obsidian Plugin API
- esbuild 构建
- Web Audio API（提示音合成）
- calendar-extended（嵌入式日历）
